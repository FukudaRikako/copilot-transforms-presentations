マウスオーバーで
強調表示

話しながら「ここ大事です」と言う代わりに、
マウスを乗せるだけ。
伝えたい言葉が自然に大きくなり、
聴衆の視線が集まる。

今すぐ試せる — このスライド上でマウスを動かして
  • 気になる行にマウスを乗せると文字が大きくなる
  • 行の背景がオレンジになり「ここが重要」と伝わる
  • 1行ずつ反応するから、ピンポイントで届く
2
🔦

スポットライト
モード

「ここを見てください」を、
言葉なしで伝える。
S キーを押した瞬間、
マウスの周囲だけが照らされ、
他はすべて暗転する。

今すぐ試せる — キーボードの S を押して
  • S キーでスポットライト ON / OFF
  • マウスを動かすと光が追いかけてくる
  • 聴衆の視線を一点に集中させられる
  • 暗転することで、言葉より強く注目を促せる
  • もう一度 S を押すと元の画面に戻る

多言語切り替え

右上のボタンひとつで日本語と英語を切り替え。
HTMLファイルは1つだけ。
翻訳データを内蔵し、
ページを再読み込みすることなく瞬時に切り替わる。

今すぐ試せる — 右上の「日本語 / EN」ボタンを押して
  • 1つのHTMLに日本語・英語の全テキストを内蔵
  • ボタンを押すだけで全スライドが瞬時に切り替わる
  • 翻訳追加も辞書にテキストを足すだけ、構造の変更は不要

フィードバック
メモ

プレゼン中に気づいた修正点を、
その場でスライドごとにメモ。
終わったらテキストをダウンロードして、
そのまま GitHub Copilot への指示に使える。

今すぐ試せる — 右上の 📝 ボタンを押して
  • プレゼン中にスライドごとのメモを即座に記録
  • メモにはスライド番号とタイムスタンプが自動付与
  • テキストでダウンロード → そのまま Copilot に貼って修正指示
